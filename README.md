# Media Manager Simulator

A web-based simulation game where you manage an online media outlet. Similar to Reigns, you'll make decisions that affect various aspects of your media company, including:
- Budget
- Reader Trust
- Reader Count
- Hidden metrics

## Features
- Dynamic decision-making system
- Chain reactions between events
- Multiple endings
- Easy to extend with new situations

## Setup
1. Clone this repository
2. Open `index.html` in your web browser
3. Start managing your media empire!

## Development
The game is built using vanilla JavaScript, HTML, and CSS. To add new situations:
1. Edit `situations.js`
2. Add new situation objects following the existing format
3. Add any new metrics to the game state in `game.js`
